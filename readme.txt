== Description ==
SocialMediaFeedWidget is a super cool widget plugin. You can use the plugin to display your Facebook Page timeline in any sidebar on your website. 

The plugin is lightweight and super fast.

Features include:

* Show/Hide Small Header.
* Show/Hide Friends & Likes.
* Show/Hide Cover photo.
* Display facebook Page timeline in any sidebar.

If you would like your website visitors to like and share your Facebook posts with ease, SocialMediaFeedWidget is perfect for you!


== Installation ==
After downloading the ZIP file of the plugin,

1. Log in to the administrator panel.
2. Go to Plugins Add-> New -> Upload.
3. Click \"Choose file\" and select the downloaded zip file of the plugin.
4. Click \"Install Now\"
5. Click \"Activate Plugin\"

== Frequently Asked Questions ==
How do I add my Facebook feed?

* Go to Appearance, Widgets, Social Media Feed Widget and drag the widget in your sidebar.
* Click on the widget and complete the fields.
* Click Save

Why does the feed not show in my sidebar?

* Please make sure that your Facebook page is not restricted.
